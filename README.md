WClouds Installation
Teil 1 (Frontend)
Gehe auf https://github.com/JW-2704/WClouds/releases/tag/v1.0.0 (Oder dem neusten Release wenns neuere gibt)
Installiere den Installer 🤯
Dann im Installer gewöhnlich ankreuzen was man will und fortfahren, ABER auf gar keinen Fall die File Explorer Integration ankreuzen
(Sie macht den File Explorer sehr laggy und macht bisschen weirdes Zeugs)
und BOOM du hast das Frontend

Teil 2 (Backend, Selbst Gehostet)
Hier kommt der schwierige Part
Gehe auf https://github.com/111111162karan/WClouds-Backend 
und lade den Source Code runter
danach mit Pycharm das Projekt öffnen und ein Venv erstellen
Dann auf requirements und diese auch installieren ins Venv
Jetzt init ausführern und auswählen was du machen willst
(Wenn man einen Key erstellt muss man selber noch eine Textdatei erstellen mit dem jeweiligen Key der ausgegeben wird)
Nun kannst du main.py starten und dich im Frontend anmelden bzw registrieren
